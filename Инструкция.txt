1. Скачать Intellij IDEA по следующей ссылке: https://www.jetbrains.com/idea/download/download-thanks.html?platform=windows

2. Запустить скачавшийся установщик, и установить программу следуя инструкциям установщика.

3. Скачать Apache Tomcat по следующей ссылке http://www-us.apache.org/dist/tomcat/tomcat-9/v9.0.5/bin/apache-tomcat-9.0.5-windows-x64.zip

4. Открыть скачавшийся архив программой WinRAR или WinZip, и распаковать содержимое в корень системного диска C:\

5. Скачать MySQL Installer по следующей ссылке: https://dev.mysql.com/get/Downloads/MySQLInstaller/mysql-installer-community-5.7.21.0.msi

6. Запустить установщик, выбрать тип установки "Developer Default", в окне конфигурации установить Port Number на "3306", MySQL Root Password - на "root".

7. В папку tomcat/lib положить файл mysql-connector-java-8.0.8-dmr.jar

8. В файле tomcat/conf/server.xml заменить следующие строки:


<Realm className="org.apache.catalina.realm.LockOutRealm">
	<Realm className="org.apache.catalina.realm.UserDatabaseRealm" resourceName="UserDatabase"/>
</Realm>


на вот эти:


<Realm className="org.apache.catalina.realm.LockOutRealm">
	<Realm className="org.apache.catalina.realm.JDBCRealm"
                   driverName="com.mysql.cj.jdbc.Driver"
                   connectionURL="jdbc:mysql://localhost:3306/pc_hardware?serverTimezone=UTC&amp;useSSL=true&amp;user=root&amp;password=root"
                   userTable="users" userNameCol="username" userCredCol="password"
                   userRoleTable="user_roles" roleNameCol="role">
                <CredentialHandler className="org.apache.catalina.realm.MessageDigestCredentialHandler" algorithm="md5" />
	</Realm>
</Realm>

9. Открыть Пуск - MySQL - MySQL Workbench, открыть подключение к БД под названием Local Instance MySQL Router. При необходимости ввести пароль "root".

10. Слева на панели Navigator нажать кнопку Startup/Shutdown, если статус сервера - "stopped", нажать Start Server.

11. Вверху, в строке меню нажать Server - Data Import. В разделе Import Options выбрать "Import from Self-Contained File". Справа нажать на кнопку "..." и в открывшемся окне выбрать файл Dump20180305.sql. Нажать Start Import.

12. Запустить Intellij IDEA, нажать "Open", в открывшемся окне выбрать папку с проектом "PCHardware".

13. При запуске внизу справа может появиться окно "Maven projects need to be imported". В окне следует нажать "Enable Auto-Import".

13/1. Если окно не появилось, откройте File - Settings - Bulid, Execution, Deployment - Build Tools - Maven - Importing, отметьте чекбокс под названием Import Maven projects automatically, и нажмите OK.

В файле other.properties хранятся номер порта MySQL, user и password, впишите свои значения если необходимо.

При разрешениях экрана ниже 1600х900 может поплыть вёрстка, если не уменьшить масштаб браузера.
